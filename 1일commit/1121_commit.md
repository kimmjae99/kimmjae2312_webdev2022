## 함수 호출 방식과 this 바인딩

- this 바인딩은 함수 호출방식, 즉 함수가 어떻게 호출되었는지에 따라 동적으로 결정된다.

### 렉시컬 스코프 VS this 바인딩

- 함수의 상위 스코프를 결정하는 방식인 렉시컬 스코프는 함수 정의가 평가되어 함수 객체가 생성되는 시점에 상위 스코프를 결정한다.
- 하지만 this 바인딩은 함수 호출 시점에 결정된다.

### 함수를 호출하는 방식

- 일반 함수 호출
- 메서드 호출
- 생성자 함수 호출
- Function.prototype.apply/call/bind 메서드에 의한 간접호출
